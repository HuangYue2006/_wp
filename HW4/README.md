# JavaScript 練習作業

## 📌 專案介紹
這是一份 JavaScript 基礎練習，共包含 10 個程式，涵蓋以下概念：

- if 判斷
- for 迴圈
- while 迴圈
- function 函式
- array 陣列
- object 物件
- JSON 處理

---

## 📂 檔案內容

### 1. 成績判斷
使用 if 判斷分數是否及格

### 2. 1~100 加總
使用 for 迴圈計算總和

### 3. 倒數計時
使用 while 進行倒數

### 4. 函式計算
建立 add 函式進行加法

### 5. 陣列加總
計算 array 所有元素總和

### 6. 物件使用
練習 object 資料存取

### 7. JSON 解析
使用 JSON.parse 轉換資料

### 8. 陣列 + 物件
輸出學生清單

### 9. function + while
計算 1~n 總和

### 10. 綜合應用
結合 if、for、function、array、object、JSON

---
